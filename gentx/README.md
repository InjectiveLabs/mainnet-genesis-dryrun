# `gentx`

To generate your `gentx` run the following command with the launch genesis file at `~/.injectived/config/genesis.json`:

```
injectived gentx \
  <$VALIDATOR_KEY_NAME> \
  --amount <amount_of_delegation_inj> \
  --commission-rate <commission_rate> \
  --commission-max-rate <commission_max_rate> \
  --commission-max-change-rate <commission_max_change_rate> \
  --pubkey <consensus_pubkey> \
  --ip=<ip_address> \
  --moniker=<name> \
  --output-document=external-val.json 
```

This will produce a file in the `~/.injectived/config/gentx/` folder that has a name with the format `gentx-<node_id>.json`. The content of the file should have a structure as follows:

```json

        "pub_key": {

```

To submit your `gentx` for inclusion in genesis, open a pull request against this repository and place the contents in a file `/gentx/<moniker>.json`.

__**NOTE**__: If you would like to override the memo field use the `--ip` and `--node-id` flags for the `gaiad gentx` command above.
